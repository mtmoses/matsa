# matsa
a Go Language library that includes collections, statistics, sets, and linear algebra.

This was created to fill the need to have a way to calculate and manipulate lists (Go Slices) with chaining capability. The library includes:

<h2>Collections</h2>
<ul>
<li>Map</li>
<li>Reduce</li>
<li>Filter</li>
<li>Reject</li>
<li>All</li>
<li>Any</li>
<li>Contains</li>
<li>Shuffle</li>
</ul>

<h2>Statistics</h2>

<li>Mean</li>

<li>HarmonicMean</li>
<li>Median</li>
<li>Mode</li>
<li>StandardDeviation</li>
<li>Variance</li>
<li>Percentile</li>
<li>Quartile</li>
<li>InterQuartileRange</li>
<li>Correlation</li>
<li>Pearson</li>
<li>Covariance</li>
</ul>
